Cherry
======

![Cherry](https://raw.githubusercontent.com/Zane-XY/cherry/master/data/cherry-128.png)

Cherry is a Firefox addon to collect text selection and POST them remote web service.

#### build & test
- `cfx test` quick check
- `cfx run` quick run
- `cfx xpi` build
